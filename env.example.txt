SANITY CMS (powers the /studio admin where Kami manages everything)

Project "Wreath Whimsy" — id 5t0ebbj0, dataset "production".

  NEXT_PUBLIC_SANITY_PROJECT_ID=5t0ebbj0
  NEXT_PUBLIC_SANITY_DATASET=production
  NEXT_PUBLIC_SANITY_API_VERSION=2024-01-01

  # Secret — server-side only. Lets the website save incoming orders into the
  # Studio. Create more tokens at sanity.io/manage (Editor permission).
  SANITY_API_WRITE_TOKEN=sk...

ORDER EMAILS (optional)
────────────────────────────────────────────────────────────────────────────
Add a Resend key to email orders to Kami automatically. Without it, the order
form falls back to opening the customer's email app — nothing is lost.

  # RESEND_API_KEY=re_...
  # ORDER_FROM_EMAIL="Wreath Whimsy <orders@yourdomain.com>"
